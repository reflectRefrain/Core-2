Glossary of Common Terms & Acronyms
  • AC – Alternating Current
  • AES – Advanced Encryption Standard
  • BIOS – Basic Input/Output System
  • BYOD – Bring Your Own Device
  • DLP – Data Loss Prevention
  • IDS – Intrusion Detection System
  • IP – Internet Protocol (also an Internet Protocol address contextually)
  • IPS – Intrusion Prevention System
  • MMC – Microsoft Management Console
  • NFC – Near-Field Communication
  • NTFS – NT File System (Windows file system format)
  • PBQ – Performance-Based Question
  • RAM – Random Access Memory
  • RDP – Remote Desktop Protocol
  • RSA – A public-key encryption technology (Rivest–Shamir–Adleman)
  • SFC – System File Checker (Windows utility)
  • SSH – Secure Shell (encrypted network protocol)
  • TPM – Trusted Platform Module
  • UTM – Unified Threat Management (all-in-one security appliance)
  • VPN – Virtual Private Network
  • WPA2 – Wi-Fi Protected Access 2 (wireless security standard)
